# 基于 ES 的大数据加速器——指标系统

## 系统介绍
多维分析
数据量：维度的笛卡尔积
空间换时间：存储换计算时间


指标概念：
- 指标对象：客户、股票、理财产品，渠道经理
- 指标：标签、属性、维度、度量值
- 容器：存储指标

系统要求
- 超级标签系统，标签可以无限扩展，所有标签可联合 filter
- cube，无限制扩展维度和度量值
- 毫秒级，多维分析同居，用户体验好，查询快
- 水平扩展，分布式，开源
- 架构部署简单，维护成本低
- 支持 SQL 查询，入门快

### ES 介绍
